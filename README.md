# Project-3--MLP

Code Features: MLP Classifier
Metrics Used and Implemented:
  Accuracy,
  Precision, 
  Recall,
  F1 Score,

How to run:

1. Create a Python virtual environment: python3 -m venv venv source venv/bin/activate

2. Install dependencies: pip install -r requirements.txt
   
3. Run the pipeline (downloads dataset): make run
